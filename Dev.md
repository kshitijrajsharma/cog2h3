## Installation 

- Install precommit
``` shell
pip install pre-commit
```
- Install dev dependencies 
```shell
pre-commit install
```

- Run Precommit against all files 
```shell 
pre-commit run --all-files
```

- Install locally 
```shell
pip install -e . 
```

- Start Developing ! 


## Versioning 

- Bump 
```shell
cz bump
```

- Push
```shell
git push
```

- Push tags
```shell
git push --tags
```
